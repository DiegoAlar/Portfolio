Welcome to iMagic!
A image is worth a thousAnd of words

What we do


We know that many times words are not enough, so we translate images into recommended Skillshare coursers so you can transform those pictures into dreams that came true.

Read Art


Our software is able to read pieces of art and find which technique is the art about so the proper class is recommended to you.

Animations


The software understands about differences in animations and with a picture it can find the course related to the animation.

Graphic Design


The software is able to recommend courses related to different graphic design techniques.

About

iMagic is a software development project meant to create a product feature to be used by creative learners. This feature uses deep learning technology to analyze images and determine which Skillshare courses better fit the properties of that picture. The team behind this project is integrated by students of software development at Holberton School along with experienced mentors from Skillshare.
